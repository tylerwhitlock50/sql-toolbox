# Domain Organization

Use one folder per business area. Keep each query where the primary business user would expect to find it.

- `gl`: Financial accounting and GL postings
- `sales`: Orders, customers, revenue operations
- `supply_chain`: Inventory, planning inputs, part flow
- `purchasing`: PO and vendor purchasing workflows
- `scheduling`: Capacity and production scheduling
- `shipping`: Packlist, shipment, fulfillment outputs
- `diagnostics`: Data-quality and system/integration diagnostics
